Project Overview

The "Baidu Public Transit Tycoon" app, developed by Baidu, offers a unique way to earn money while using public transportation. Users complete simple tasks during their daily bus rides, such as taking photos and checking in, to receive rewards and save on commuting costs. Additionally, the app provides real-time bus route information and rewards users for submitting accurate data about bus stops and stations across China.

Role and Responsibilities

  • Developed the task list module to display current city's public transit tasks, allowing users to filter tasks based on criteria such as distance and points.
  • Constructed the task collection interface, enabling users to capture photos and data at initial, final, and midpoint stations, with features like station management.
  • Implemented the task upload functionality, enabling users to submit completed tasks for backend review
  • Designed the backend review system to ensure the legitimacy of submitted task data, rewarding points upon successful verification

Technologies and Tools Used

  • Back-end: PHP, Yaf framework, PostgreSQL, Kafaka
  • Version Control: Git
  • DevOps: Baidu cloud, CI, CD pipeline

Project Challenges and Solutions

  • Challenge: Task Filtering and Management.
  • Solution: Addressing the need for efficient task management, we developed a task list module that allowed users to filter tasks based on distance and points. This streamlined users' ability to find tasks that matched their preferences.
  • Challenge: Data Integrity and Security.
  • Solution: Compress, package, and slice large-sized images to improve upload success rate and user experience. In addition, we employed encryption techniques helped safeguard sensitive information.
  • Challenge: Real-Time Map Integration.
  • Solution: We integrated real-time map services. This provided users with a dynamic visual representation of their progress within the app.

Project Goals and Achievements

  • The rewarding of points and their conversion into cash has significantly boosted user engagement, ensuring sustained user activity over the long term. This approach has also enhanced the efficiency and accuracy of maintaining our public transit routes and stations, ultimately providing users with more precise and convenient data.
  • Recommend local tasks based on the user's proximity, making it easier for users to participate in tasks.

Team Collaboration and Communication

  • Throughout the project, I collaborated closely with cross-functional teams, discussing requirements with product managers, coordinating interface design with designers, and troubleshooting issues with testing engineers.
  • Through ongoing communication and collaboration, we were able to address challenges promptly and ensure the project's progress as planned.

Self-Assessment

In this project, I not only enhanced my technical skills but also developed teamwork and problem-solving abilities.
